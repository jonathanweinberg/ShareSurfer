These screenshots capture the current standalone dashboard QA views from June 9, 2026. They are retained for documentation, release notes, and first-run user guidance.

The screenshots use demo/export data only and do not contain customer evidence.

## Usage

Use these images when explaining what the standalone dashboard looks like before a first-time operator runs ShareSurfer. Prefer the newest screenshot set when multiple dated folders exist.
